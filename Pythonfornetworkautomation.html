Working with text files

Open the file as readonly mode 
f = open('filename','r') # open the file as readonly
	print(f) # print the whole content of a file
f.close() # close the file
print(f.closed) # check if the file is closed
A more "standardized way" 
with open('filename','r') as file:
	#do something with your text
	print(file.read())
# you don't have to close the file, it'll automatically close the file for you

Writing to a file
code
 
with open('a.txt','w') as f:
	f.write('MyFirstLine\n')
	f.write('MySecondLine')	
# Please keep in mind that 'w' mode will overwrite the whole file, if a file doesn't exist, it'll create a new file.
# Allow write only
			
with open('a.txt','a') as f:
	f.write('\nMyLastLine')
# Please keep in mind that 'a' mode will continue to write at the end of file(EOF)
# Allow write only
			
with open('a.txt','r+') as f:
	f.write('This is a line added by r+')
# Read & Write
# Write at the beginning of file, overwrite the beginning content. Can use seek()
# Does not create a file if it doesn't exist.
